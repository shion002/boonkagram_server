# Boonkagram

<br/>

- <b>소개</b> : 분위기 좋은 카페를 모아놓은 데이터 기반 원하는 카페를 찾고, 관리자 페이지를 통한 유지보수
- <b>Frontend Skill</b> : React, TypeScript
- <b>Backend Skill</b> : Spring boot JPA, PostgreSQL, Nginx
- <b>주요 기능</b> : 카페 조회, 로그인, 리뷰 작성, 검색, 위치기반 검색, 관리자 페이지 (카페 추가, 수정, 삭제)
- <b>프론트 Repository</b> : https://github.com/shion002/boonkagram
